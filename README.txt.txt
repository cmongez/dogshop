Vamos a crear un sitio web para “dogShop”, donde permita mostrar la información de la empresa,
se le solicitará:
● Construir estructura HTML utilizando etiquetas semánticas, mínimo 3, como header,
section, footer, nav o article.
● Crear un archivo Css.
● Agregar imágenes al proyecto.
● Realizar el proyecto bajo la metodología BEM.
● La página debe tener títulos y subtítulos para las secciones.
● Se deben usar 1 o 2 tipos diferentes de fuentes.(Google fonts).
● La página debe tener una barra de navegación.
● Añade un footer con “Todos los derechos reservados” centrado color de fondo negro ,
letras color blanco.
